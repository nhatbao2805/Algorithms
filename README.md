# 📚 Ôn Luyện Thuật Toán JavaScript

Bộ tài liệu ôn thuật toán từ cơ bản đến nâng cao bằng JavaScript.

- **algorithms/** – 25 file thuật toán (Big-O, tìm kiếm, sắp xếp, DP, đồ thị, …)
- **algorithms/playground.js** – File code thử nghiệm (chạy: `npm run playground`)

Chi tiết xem [algorithms/README.md](algorithms/README.md).

## Chạy nhanh

```bash
npm install
node algorithms/00-big-o-notation.js
npm run playground   # tự chạy lại khi save playground.js
```
